# 文档02：硬件设计说明书
## 1. 硬件架构框图
```
12V DC输入 → 5A保险丝 → 1N5408反接保护 → SMBJ15CA浪涌保护 → 分两路：
                                            ↓ 12V → 4路电磁阀驱动电路（每路带SS34续流二极管+RC吸收）→ 电磁阀
                                            ↓ MP2307 DC-DC降压到5V → 分两路：
                                                                      ↓ 5V → 流量传感器、LCD、编码器等外设
                                                                      ↓ ME6211 LDO降压到3.3V → ESP32主控、RTC、数字电路
```

## 4. PCB设计规范（4层板）
### 4.1 层叠结构
| 层号 | 类型 | 说明 |
|------|------|------|
| 1 | 信号层 | 顶层，走信号线，放主要元件 |
| 2 | GND层 | 完整地层，无分割 |
| 3 | POWER层 | 电源层，包含12V、5V、3.3V分割 |
| 4 | 信号层 | 底层，走信号线 |

### 4.2 布局要求
1. 电源部分布局在PCB左侧，按输入→保护→降压的顺序布局，远离高频电路
2. ESP32模组布局在PCB中间，天线部分朝向PCB边缘，净空≥15mm，无走线无敷铜
3. 4路电磁阀驱动电路布局在PCB右侧，靠近接线端子，每路独立分区，避免干扰
4. 人机交互元件（按键、LED、LCD接口）布局在PCB上方，对齐外壳开孔
5. 所有对外接口布局在PCB边缘，方便接线

### 4.3 布线要求
1. 12V电源走线宽度≥3mm，5V走线≥2mm，3.3V走线≥1mm
2. GND大面积敷铜，所有层空余区域全部敷铜接GND，过孔密度≥1个/平方厘米
3. I2C信号线等长，长度≤10cm，远离高频走线
4. 流量传感器输入走线走短线，增加100nF滤波电容靠近接口
5. 电磁阀驱动回路走线尽量短，减少压降
6. 所有对外接口信号引脚串联100Ω电阻，并联ESD保护二极管

### 4.4 丝印要求
1. 所有接口标注引脚定义：VCC、GND、SDA、SCL、OUT1、OUT2等
2. 电磁阀、流量传感器接口标注对应路数：ROAD1、ROAD2等
3. 按键标注功能：STOP、START、RESET等
4. 标注PCB版本号、设计日期、丝印清晰易读

## 5. 防护&EMI抗干扰设计要求
1. 基础防护：
   - 电源输入端：5A保险丝、1N5408反接保护、SMBJ系列TVS浪涌保护
   - 所有对外接口：ESD保护二极管，防止静电损坏
   - PCB焊接完成后整体刷三防漆，防潮、防盐雾、防霉
   - 外壳选用IP65防水等级，所有对外接线采用防水接头
2. 电磁阀EMI抗干扰设计：
   - 每路电磁阀两端并联RC吸收电路（100Ω电阻+104电容串联），抑制开关尖峰
   - 续流二极管SS34/SS54尽量靠近电磁阀接线端子放置，缩短回流路径
   - 电磁阀驱动走线和敏感信号线（流量传感器、I2C、编码器）间距≥5mm，避免交叉
   - 12V电源输入端增加100μF电解电容滤波，抑制传导干扰
   - 金属外壳建议连接到系统GND，屏蔽辐射干扰
3. 走线优化设计：
   - 4路电磁阀驱动引脚GPIO13/14/15/16位于ESP32右侧，直接连接到右侧电磁阀接口，无交叉走线
   - I2C引脚GPIO21/22位于ESP32左侧，直接连接到LCD和RTC接口，走线最短
   - 流量传感器输入引脚GPIO34/35/36/39位于ESP32上方，直接连接到左侧流量传感器接口
   - 按键和编码器引脚位于ESP32下方，直接连接到正面人机交互接口，无需跨层走线

## 6. 外壳安装要求
1. PCB通过M3铜柱固定在外壳内部，距离底部≥5mm，避免积水
2. 外壳开孔对齐LCD、按键、LED位置，开孔处打胶防水
3. 安装位置避雨、通风，避免阳光直射PCB和元件
