# 文档06：测试验证方案
## 1. 测试环境
- 硬件：成品控制器、12V 2A电源、4个12V电磁阀、4个流量传感器、水龙头、量杯
- 软件：串口调试工具、点灯科技APP、Web浏览器
- 环境：常温、2.4G WiFi网络

## 2. 测试用例
### 2.1 硬件测试
| 测试项 | 测试方法 | 验收标准 |
|--------|----------|----------|
| 电源测试 | 12V输入，测量5V、3.3V输出电压 | 电压误差≤±5% |
| 电磁阀驱动测试 | 控制每路电磁阀开关 | 每路都能正常开启关闭，无卡顿 |
| 流量传感器测试 | 水流过传感器，查看脉冲计数 | 1L水脉冲数≈450，误差≤±10% |
| LCD显示测试 | 开机查看所有界面 | 显示清晰，无缺行少字 |
| 按键测试 | 按所有按键、编码器旋转 | 所有按键响应正常，编码器无跳步 |

### 2.2 功能测试
| 测试项 | 测试方法 | 验收标准 |
|--------|----------|----------|
| 手动浇水测试 | 设置时长和路数，启动浇水 | 对应电磁阀开启，到点自动关闭，记录正常 |
| 定时任务测试 | 新建每天模式任务，到点触发 | 准时开启浇水，时长正确，记录正常 |
| 紧急停止测试 | 浇水中按红色键 | 所有电磁阀立即关闭，记录异常停止 |
| 流量校准测试 | 放水1L，傻瓜校准 | 校准后计量误差≤±5% |
| 按键锁定测试 | 长按编码器锁定 | 按键失效，紧急停止仍有效，长按解锁恢复 |
| 恢复出厂测试 | 触发两种重置模式 | 配置恢复默认，记录按选择保留/清空 |
| WiFi连接测试 | 配置WiFi密码 | 1分钟内连接成功，APP和Web可访问 |
| 休眠测试 | 无操作10分钟后 | 功耗下降到≤2mA，Web访问仍正常 |
| 异常告警测试 | 开启浇水后关闭水源 | 2分钟内检测到水流异常，关闭电磁阀，LED快闪 |

### 2.3 稳定性测试
| 测试项 | 测试方法 | 验收标准 |
|--------|----------|----------|
| 连续运行测试 | 设备通电连续运行72小时，每天触发3次浇水 | 无死机、无重启、无漏浇水、无误触发 |
| 压力测试 | 连续100次手动启停浇水，频繁修改配置 | 无死机、无数据丢失、配置保存正常 |
| 掉电测试 | 浇水中突然断电，重新上电 | 数据不损坏，上电后恢复正常，记录完整 |

## 3. 验收标准
所有测试用例100%通过，无功能性BUG，连续72小时运行稳定，流量校准后误差≤±5%，即可交付。
