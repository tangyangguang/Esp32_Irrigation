# 07 Web 页面内容与交互草案

## 总览

区块：

- 系统状态：主控时间、从站数量、当前异常数。
- 当前运行摘要：正在运行的水源、水路、剩余时间、估算水量。
- 水源概况：每个水源的在线、运行、禁止启动、异常状态。
- 最近事件：最近几条需要注意的事件。
- 今日概况：今日运行次数、总时长、估算总水量。

交互：

- 只保留查看摘要所需的操作。
- 跨页面移动使用顶层导航，不在区块内重复设计快捷入口。

## 运行

区块：

- 当前任务：每个正在运行水源的任务卡片。
- 手动启动：选择水源、水路、时长。
- 启动阻断：显示低液位、漏水、离线、忙碌等原因。
- 保护状态：当前水源的流量、低液位、漏水状态。

交互：

- 启动手动浇水。
- 停止正在运行的任务。
- 查看阻断原因。

限制：

- 不编辑计划。
- 不修改硬件映射。
- 不修改系统参数。

## 计划

区块：

- 计划列表：名称、启用状态、下一次触发、步骤摘要。
- 筛选或状态切换：全部、启用、停用。
- 分页：计划数量增加时使用。
- 计划编辑视图：基础信息、每天或自定义周期、开始时间、水路步骤。

说明：

- 计划列表页不在列表旁边或下方直接铺编辑表单。
- 新建、编辑计划进入独立编辑页；保存或取消后返回计划列表。

交互：

- 新建计划。
- 编辑计划。
- 启用/停用计划。
- 添加、删除、排序步骤。

限制：

- 不显示浇水记录明细。
- 不处理事件。
- 不直接启动普通手动浇水。

## 水源

区块：

- 水源列表：名称、从站地址、在线状态、运行状态、异常状态。
- 水源详情：主阀、水泵、流量计、低液位、电压读数。
- 水路列表：名称、启用状态、参考流量、最近运行。
- 校准状态：参考流量是否已校准。
- 水源配置视图：主阀阀号、水泵启用、流量计输入、低液位输入、阀 PWM 参数。
- 从站发现和配置状态：未配置从站、地址异常、配置不完整。

交互：

- 查看水源详情。
- 查看水路详情。
- 进入水路校准。

限制：

- 不作为普通手动浇水入口。
- 不在水源列表下方或旁边编辑低频硬件映射；硬件映射进入水源详情下的独立配置页面。

## 记录

区块：

- 记录列表：时间、水源、水路、来源、时长、水量、稳定流速、结束原因。
- 筛选：时间、水源、水路、来源、结束原因。
- 分页：记录默认按页展示。
- 记录详情视图：开始结束、计划时长、实际时长、脉冲数、估算水量、稳定流速、保护触发。

交互：

- 查看记录详情。
- 按条件筛选。
- 对正常完成的记录执行“采用为参考流量”。

限制：

- 不显示未运行的跳过项。
- 不处理事件。
- 不发起浇水。
- 不在列表下方或旁边铺详情或表单；详情进入独立详情页。

## 事件

区块：

- 当前事件：仍在发生或仍影响启动的事件。
- 历史事件：已恢复或信息类事件。
- 筛选：来源分类、影响级别、状态、水源、时间。
- 分页：事件默认按页展示。
- 事件详情视图：时间、水源、水路、原因、影响、恢复时间。

事件列表状态字段：

- 发生中。
- 已恢复。
- 已记录。

交互：

- 查看事件详情。

限制：

- 不把事件当作浇水记录。
- 不在事件页发起普通浇水。
- 不在列表下方或旁边铺详情或表单；详情进入独立详情页。

## 系统参数

区块：

- 全局运行参数：最大单次运行时长、主阀和水泵启停时间。
- 通信参数：RS485 通信速率、从站轮询周期、离线失败次数。
- 流量保护参数：启动宽限、无流量确认、漏水窗口、低/过流量比例和动作。

交互：

- 修改系统级参数。
- 保存或取消。

运行中限制：

- 系统参数修改只影响新任务或后续判断，不重写已经展开的任务。

限制：

- 不发起普通浇水。
- 不展示完整记录列表。
- 不做通用诊断后台。
- 不显示水源级参数。水源名称、主阀、水泵、流量计、低液位、阀 PWM 在水源详情下的配置视图修改。
- 不显示水路级参数。水路名称、启用状态、参考流量在水路详情下的配置视图修改。

## 校准

校准不是顶层页面，作为水路详情中的操作进入。

校准用途：

- 运行一次指定水路。
- 采集稳定流量。
- 写入该水路参考流量。

校准规则：

- 遵守低液位、漏水、无流量保护。
- 同一水源运行中不能校准其它水路。
- 校准结果只影响低/过流量判断，不改变计划时长。
