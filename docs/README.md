## 当前规则

- 只把 `pcb/` 当前文件作为硬件事实依据。
- `历史参考/` 只作为参考，不作为结论依据。
- 已归档的过程性原型不作为参考，不作为当前方案入口。
- 当前 Web 原型用于验证页面结构、任务流、信息层级和视觉表达。
- 页面内容必须使用真实逻辑、默认值、空状态或待配置状态，不使用虚构设备和虚构记录。
- 原型确认前不进入固件、API、存储、通信协议和工程实现设计。
