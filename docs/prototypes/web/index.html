天气暂停

只影响自动计划。触发后到点计划跳过，不补跑；手动浇水仍可使用。
单位：当天时间。天气暂停结束后，自动浇水恢复为已开启。
单位：%。预报降雨概率达到或超过该值时，可触发天气暂停。
单位：毫米。预计降雨量达到或超过该值时，可触发天气暂停。
单位：米/秒。风太大时喷洒容易飘散，可暂停自动计划。
单位：摄氏度。低于该温度时暂停自动计划，避免低温浇水风险。
天气暂停是自动计划的保护条件，不会改变每路浇水时长，也不会按天气自动增减水量。

自动和外设

单位：分钟。自动计划撞上当前运行时，最多等待多久。
计划排队最长等待设为 0，表示遇到冲突时不排队，直接跳过。

故障策略

固定规则：无水一定停机；低液位一定停机但不锁定水路；待机漏水和流量计异常一定全局锁定。

高级：流量检测参数

通常保持默认值；只有排查误报或现场调试时才需要修改

⌄
单位：秒。用最近这段时间的脉冲计算实时流速。
单位：毫秒。页面和运行监测多久刷新一次流速。
单位：秒。开阀后多久还没有脉冲就判断无水。
单位：秒。稳定运行后连续多久无脉冲就停机。
单位：秒。首次出水后先等待水流稳定，期间不判断偏低/偏高。
单位：秒。校准和测定标准流速时的最长放水时间。
单位：秒。空闲时连续检测到流量多久后锁定设备。

高级：阀门保持参数

会影响阀门吸合、发热和噪声，修改后需要实机验证

⌄
单位：毫秒。阀门刚打开时保持全功率的时间。
单位：%。吸合后用较低功率保持打开；100% 表示不降功率。
单位：赫兹。保持阶段 PWM 频率，需要结合阀门实测。
阀门保持功率比例设为 100，表示不降低保持功率。修改阀门参数需要实机验证。
