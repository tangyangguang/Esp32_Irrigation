天气自动暂停计划

启用后，设备会根据天气预报自动把“自动浇水”暂停到下次复核时间。只影响自动计划，手动浇水仍可使用。
到这个时间重新获取天气并判断是否恢复。若预报仍达到阈值，会继续暂停到下一次复核。
系统查看未来多少小时内的降雨、风和低温风险。
预报判断范围内达到或超过该概率时，可触发天气暂停。
预报判断范围内累计雨量达到或超过该值时，可触发天气暂停。
高级天气保护 强风和低温，通常保持默认
风太大时喷洒容易飘散，可暂停自动计划。
低于该温度时暂停自动计划，避免低温浇水风险。
天气自动暂停本质上是把自动总控设为“暂停到某个时间”。暂停期间到点计划直接跳过，不补跑；它不会改变每路浇水时长，也不会按天气自动增减水量。

自动和外设

启用后，水路运行前会先启动自吸泵输出，停机时同步关闭。
启用后，检测到低液位会立即停止当前运行，但不锁定水路。
自动计划撞上当前运行时，最多等待多久。
计划排队最长等待设为 0，表示遇到冲突时不排队，直接跳过。

故障策略

固定规则：无水一定停机；低液位一定停机但不锁定水路；待机漏水和流量计异常一定全局锁定。
