# 03 配置参数与校准

## 配置原则

配置只暴露真实影响行为的参数。

不提供：

- 输入选择。
- 输出选择。
- 主阀配置。
- RS485 参数。
- 从站地址。
- 通用 IO 配置。

## 系统参数

系统参数影响所有任务：

| 参数 | 默认值 | 说明 |
| --- | ---: | --- |
| 最大单次运行时长 | 120 分钟 | 手动和计划都受限 |
| 阀全功率吸合时间 | 待定 | 打开阀后先 100% 输出 |
| 阀维持占空比 | 待定 | 吸合后 PWM 维持 |
| 自吸泵启动延时 | 1 秒 | 启用自吸泵时使用 |
| 自吸泵停止延时 | 1 秒 | 启用自吸泵时使用 |
| 历史保留数量 | 待定 | 防止存储无限增长 |

阀驱动参数是全局参数，不逐分区配置。

## 供水配置

供水配置影响整套设备：

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| 自吸泵控制 | 禁用 | 启用后使用 `H7` 干接点输出 |
| 低液位传感器 | 禁用 | 启用后使用 `H4` 干接点输入 |
| 低液位触点类型 | 常开 | 可选常开或常闭 |
| 低液位去抖时间 | 1 秒 | 防止抖动误停 |
| 流量计每升脉冲数 | 未设置 | 接水量校准后生成或手动录入 |

流量计是必配固定输入，因此没有启用项，也没有输入选择项。

## 分区配置

每个分区固定对应一路阀输出。

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| 分区名称 | 分区 1..6 | 用户可改名 |
| 启用状态 | 启用 | 停用只阻止新任务 |
| 标准流量 | 未设置 | 稳定流量校准后生成 |
| 最近手动时长 | 无 | 有历史后可复用 |

分区不配置阀号。阀号由硬件顺序固定：

- 分区 1 -> 阀 1。
- 分区 2 -> 阀 2。
- 分区 3 -> 阀 3。
- 分区 4 -> 阀 4。
- 分区 5 -> 阀 5。
- 分区 6 -> 阀 6。

## 计划配置

计划包含：

- 名称。
- 启用状态。
- 周期类型。
- 开始时间。
- 分区步骤。

周期类型：

- 每天。
- 自定义周期。
- 每周固定日。

分区步骤包含：

- 分区。
- 运行时长。
- 启用状态。

触发规则：

- 到点后按步骤顺序执行。
- 设备忙或保护阻止时，本次计划跳过并记录原因。
- 不排队。
- 不补跑。
- 修改计划只影响后续触发。

## 校准

校准保留两类。

### 接水量校准

目的：得到流量计每升脉冲数。

流程：

1. 用户选择一个分区。
2. 系统检查设备空闲、分区启用、低液位状态。
3. 系统启动该分区并统计脉冲。
4. 用户停止或到达设定时长后停止。
5. 用户输入实际接水量。
6. 系统计算每升脉冲数。
7. 用户保存或取消。
8. 保存后写入历史。

### 稳定流量校准

目的：得到该分区标准流量，用于低/过流量判断。

流程：

1. 用户从分区详情进入。
2. 系统启动该分区。
3. 过启动流量宽限后观察流速。
4. 流速稳定后给出建议标准流量。
5. 用户保存或取消。
6. 系统停止分区并写入历史。

## 运行中修改限制

运行中禁止修改：

- 自吸泵启用状态。
- 低液位启用状态和触点类型。
- 流量计每升脉冲数。
- 阀驱动参数。
- 正在运行分区的启用状态。

停用分区只阻止新任务，不停止当前任务。停止当前任务必须使用明确的停止操作。
