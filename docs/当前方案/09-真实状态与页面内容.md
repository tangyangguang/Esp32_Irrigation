# 09 真实状态与页面内容

## 目标

Web 界面方案不使用虚构设备、虚构计划、虚构历史和虚构人名。

页面只能展示三类内容：

- 已确认的产品逻辑。
- 来自硬件和方案文档的默认值、范围和规则。
- 当前无真实数据时的空状态、待配置状态和不可操作原因。

页面结构必须像真实产品界面，而不是说明文档。对象列表页不要把完整编辑表单并排铺在主页面；新增、编辑、详情和校准进入弹窗、抽屉或独立详情视图。

这样做的目的不是让页面显得“有数据”，而是让信息层级、操作路径和边界判断可以被真实确认。

## 当前真实前提

在没有接入真实主控、从站和历史数据前，界面默认状态为：

- 主控：无实时连接数据。
- 从站：未发现或未配置。
- 水源：无真实水源实例。
- 水路：无真实水路实例。
- 当前任务：无。
- 自动计划：无。
- 历史：无记录。
- 异常：无实时判断。

页面可以显示这些空状态，但不能编造“某水源正在运行”“某计划已经执行”“某从站离线”等事实。

## 页面内容原则

### 总览

总览只回答系统是否有可判断状态。

无真实数据时显示：

- 当前无实时设备连接。
- 从站发现数量为 0 或显示“待接入”。
- 当前任务为空。
- 当前异常为空，但注明“无实时判断不等于系统正常”。
- 系统结构：主控只做管理中心，从站对应水源，水源下有水路。

不显示：

- 今日浇水次数。
- 今日水量。
- 正在运行的虚构任务。
- 虚构异常数量。

### 运行

运行页只回答能不能立即手动浇水。

无真实配置时显示：

- 无可启动水源。
- 手动启动表单不可用。
- 不可启动原因：未接入或未配置水源、水路和从站。
- 从站闭环规则：主控下发带时长任务，从站本地计时，到点关闭；异常时从站按本地规则停机并上报。

不显示：

- 虚构水源下拉选项。
- 虚构当前流速。
- 虚构剩余时间。

### 计划

计划页只回答未来什么时候自动浇水。

无真实计划时显示：

- 计划列表为空。
- 可新建计划结构。
- 周期只保留每天和自定义周期。
- 步骤必须绑定真实水源和水路；没有真实水源时不能形成可执行计划。

不显示：

- 虚构早晨计划。
- 虚构下一次触发时间。
- cron、脚本、天气条件。

### 水源

水源页只回答水源、从站、水路和固定硬件能力如何配置。

无真实水源时显示：

- 水源列表为空。
- 从站发现后才能形成水源实例。
- 一个从站对应一个水源。
- 一个水源同一时刻只运行一条水路。
- 同一水源只有一个流量计，当前流量归属当前运行水路。
- 主阀默认阀 8，可按水源修改。
- 自吸泵/水泵输出可启用或停用。
- 低液位传感器可启用或停用，不是必配。

输入选择显示为水源配置：

- 流量计输入：从 IN1..IN4 中选择。
- 低液位输入：从 IN1..IN4 中选择。
- 同一水源内两者不能选同一路输入。
- 这不是通用 IO 平台。

### 历史

历史页统一承载浇水、异常、配置、校准和操作。

无真实记录时显示：

- 历史为空。
- 可展示筛选结构和详情字段结构。
- 说明哪些动作未来会产生记录。

不显示：

- 虚构完成记录。
- 虚构异常记录。
- 虚构配置修改记录。

### 系统参数

系统参数页只放全局参数。

可以显示已确认的默认值和规则：

- 最大单次运行时长：默认 120 分钟。
- 主阀默认阀号：8。
- 低流量、过流量默认只提示，可配置为停机。
- 无流量保护用于缺水或流量计无脉冲场景。

不显示：

- 水源级配置。
- 水路级配置。
- 从站输入选择。
- 任意 IO 调试台。
- 任意 RS485 调试台。

## 必须覆盖的状态场景

以下是状态逻辑，不是虚构数据。

### 启用自吸泵且启用低液位

启动前低液位有效：

- 不启动任务。
- 记录异常。
- 页面显示不可启动原因。

运行中低液位有效：

- 从站停止输出。
- 记录异常停止。
- 当前任务结束原因为低液位。

### 启用自吸泵但未启用低液位

启动前：

- 不检查低液位。
- 自吸泵按任务启停。

运行中缺水：

- 依靠无流量保护停止。
- 记录无流量异常。

### 低流量或过流量

默认：

- 只提示。
- 不停机。

配置为停机时：

- 从站或主控按已确认规则停止当前任务。
- 记录异常停止。

### 从站通信中断

未启动任务：

- 主控不再启动该水源新任务。
- 页面显示不可启动原因。

已下发有时长任务：

- 从站继续按本地计时闭环执行。
- 到点关闭输出。
- 通信恢复后同步最近任务结果。

### 计划错过

到点时主控不可执行或水源不可用：

- 不补跑。
- 记录跳过或异常原因。

## 页面禁用内容

当前界面方案禁止出现：

- 虚构水源、水路、从站地址、计划、历史。
- 虚构运行数据、流量、流速、电压。
- 虚构人名、账号、操作人。
- 天气、统计独立页、OTA、LCD、本地按键、复杂权限。
- 记录页和事件页拆分。
- 通用 IO 平台。
- 复杂调试后台。
