# 当前方案

本目录定义当前唯一有效的 12V 智能浇水控制器方案。

阅读顺序：

1. `00-硬件能力与边界.md`
2. `01-产品对象与功能范围.md`
3. `02-运行规则与保护.md`
4. `03-配置参数与校准.md`
5. `04-Web与本地操作.md`
6. `05-工程实现边界.md`

用于确认功能范围的网页：

- `功能列表与边界.html`

核心结论：

- 单板控制器，不再有 RS485、主控、从站或分站。
- 只有一套供水系统，不做多水源并行。
- 6 路输出全部是水路电磁阀，不做主阀。
- 流量计必配，固定接入。
- 低液位传感器可选，固定干接点输入，可配置常开/常闭，只在自吸泵启用时参与保护。
- 自吸泵可选，固定干接点输出，只控制外部继电器模块信号。
- ESP32 固件基于 `../Esp32Base` 基础库完整能力搭建，项目代码只在其上实现浇水业务。
