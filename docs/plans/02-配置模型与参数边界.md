# 02 配置模型与参数边界

## 配置分层

| 配置类别 | 保存位置 | 用途 |
|---|---|---|
| 主控系统配置 | 主控 | 网络、时间、LCD、SHT30、RS485、记录策略 |
| 从站业务配置 | 主控 | 从站名称、水源名称、水路名称、启用状态 |
| 从站安全配置 | 主控 + 从站 | 输入/输出映射、本地安全参数、阀门参数 |
| 单次运行参数 | 主控下发，从站临时使用 | 运行时长、标准流速、流量保护参数 |

主控保存完整业务配置。从站只保存足够保证本地安全执行和安全停机的配置。

## 主控系统配置

主控系统配置包含：

- 设备名称。
- Web/API 访问配置。
- 网络配置。
- RTC 时间同步状态。
- LCD2004 显示配置。
- SHT30 温湿度显示配置。
- RS485 波特率，默认 `19200 bps`。
- 从站轮询周期。
- 事件和运行记录保留策略。

主控板不启用 FRAM。配置、计划、校准参数和记录由主控侧负责生命周期管理；具体存储介质属于实施阶段设计。

## 从站业务配置

```text
station_id
rs485_address
enabled
station_name
water_source_name
notes
```

规则：

- `rs485_address` 来自拨码开关，范围 `1..15`。
- 地址 `0` 保留。
- 主控需要检测疑似地址冲突。
- 未启用从站不参与手动运行和计划运行。

## 水路配置

每个从站最多 8 路阀输出，对应 `VALVE1..VALVE8`。

每一路阀输出角色：

- `disabled`
- `branch_zone`
- `master_valve`

水路配置：

```text
zone_index             1..8
role
enabled
name
default_duration_sec
lock_state
manual_allowed
schedule_allowed
baseline_ml_min
baseline_status
```

规则：

- 一个从站最多 1 路主阀。
- 主阀不能同时作为分支水路。
- 主阀不作为用户可直接运行水路。
- 至少 1 路分支水路启用后，从站才具备可运行水路。
- 标准流速属于分支水路，不属于主阀。

## 输入配置

从站 `IN1..IN4` 可配置为：

- `disabled`
- `flow_meter`
- `low_level`
- `reserved`

输入配置：

```text
input_index             1..4
role
active_level            low / high
debounce_ms
```

规则：

- 一个从站必须且只能配置 1 路流量计。
- 低液位可选。
- 同一路输入不能同时承担多个角色。
- `reserved` 只表示 `IN1..IN4` 中该路暂不参与当前灌溉逻辑。
- `reserved` 不生成保护、不参与运行准入、不要求传感器参数。
- 从站 `H6/H8/H9`、`SPARE_*` 不进入当前 Web 输入配置。

## 流量计配置

```text
input_index
edge                    rising / falling
pulses_per_liter
sample_window_sec
display_refresh_ms
first_flow_timeout_sec
no_flow_timeout_sec
stabilize_sec
standby_leak_confirm_sec
```

规则：

- `pulses_per_liter` 来自流量计校准。
- 计量未校准时仍允许灌溉运行，但水量估算不可用或仅供参考。
- 只要能检测脉冲，仍可做无水判断。
- `stabilize_sec` 只屏蔽低/高流量判断，不屏蔽无水和低液位保护。

## 低液位配置

```text
enabled
input_index             0=不启用，1..4=输入
active_level
confirm_ms
action                  stop_current_source
```

规则：

- 未配置低液位时，不做低液位保护。
- 配置低液位后，运行中触发并确认必须停机。
- 低液位是水源级条件，不是单一路水路特征。

## 自吸泵配置

```text
enabled
output                  dry_contact_1
start_delay_ms
stop_delay_ms
```

规则：

- 未启用时，从站不控制干接点输出。
- 启用时，运行任意分支水路都会联动自吸泵。
- 同时启用低液位时，启动自吸泵前必须先确认低液位未触发。

## 主阀配置

```text
enabled
valve_index             0=无主阀，1..8=主阀
start_delay_ms
stop_delay_ms
```

规则：

- 主阀可选。
- 未配置主阀时，分支阀可直接运行。
- 配置主阀时，启动顺序为自吸泵、主阀、分支阀；停止顺序反向或按安全停机策略关闭。

## 阀门参数

```text
valve_pull_in_ms
valve_pwm_freq_hz
valve_hold_percent
```

规则：

- 默认策略必须能在未验证 PWM 降功率时使用 100% 保持。
- PWM 降功率是增强能力，未验证不阻断运行。
- 安全停机时必须关闭 `DRV_SD` 和所有阀输出。

## 电压检测配置

```text
voltage_protection_enabled
voltage_calibrated
voltage_low_warn_mv
voltage_low_stop_mv
voltage_high_stop_mv
```

规则：

- 电压检测未校准或保护未启用时，不阻断运行。
- 未启用时只提示电压保护未启用。
- 已启用、已校准并确认严重低压或高压时，才禁止启动或触发停机。

## 配置同步规则

- 主控维护配置版本号和 CRC。
- 从站保存安全配置版本和 CRC。
- 主控轮询时检查从站配置摘要。
- 版本或 CRC 不一致时，相关从站不能运行，必须重新同步配置。
- 从站处于 `STARTING`、`RUNNING`、`STOPPING` 时拒绝写入安全配置。

## 增强能力降级原则

允许启动但必须提示保护不完整：

- 流量计计量校准未完成。
- 标准流速未学习。
- 电压检测未校准或电压保护未启用。
- SHT30、LCD、待机漏水等增强诊断或显示能力不可用。
- 主阀未配置。
- 低液位未配置。
- 自吸泵未配置。

不得把“保护未启用”显示为“设备故障”。
