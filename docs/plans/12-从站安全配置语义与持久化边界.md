# 12 从站安全配置语义与持久化边界

状态：成熟产品方案层文档。本文只定义从站安全配置的业务语义、有效性、同步关系和故障边界，不定义 EEPROM/IAP、双槽、文件格式或具体存储算法。

## 设计目标

从站安全配置解决一个核心问题：

> 主控下发运行命令后，即使主控临时离线、RS485 通信丢失或从站重启，从站也必须知道哪些输入输出是安全相关资源，以及遇到危险条件时应该关闭什么。

因此，从站安全配置不是完整业务配置，也不是 Web 展示配置。它只是从完整业务配置中投影出来的一小组“本地安全运行所必需的信息”。

## 主控和从站的配置关系

主控是完整业务配置的唯一管理者。

主控保存：

- 从站名称。
- 水源名称。
- 水路名称。
- 计划。
- 页面排序。
- 用户备注。
- 运行记录。
- 校准历史。

从站只保存或持有安全运行所需配置：

- 哪一路输入是流量计。
- 哪一路输入是低液位。
- 输入有效电平。
- 哪一路输出是主阀。
- 哪些输出是分支水路。
- 是否启用自吸泵输出。
- 启动、运行、停止需要的安全时间参数。
- 通信超时和最大运行时间。
- 电压和流量保护所需的本地判断参数。

## 安全配置有效性

从站安全配置必须能回答：

- 当前配置是否存在。
- 当前配置是否完整。
- 当前配置是否属于当前主控认为的版本。
- 当前配置是否通过字段合法性校验。
- 当前配置是否足够支持本地安全停机。

从站不能只因为“收到过配置”就认为配置有效。

配置有效至少要求：

- 有且只有一个流量计输入。
- 低液位输入不与流量计输入重复。
- 主阀不与分支水路重复。
- 至少有一路分支水路启用。
- 安全时间参数在允许范围内。
- 电压和流量保护参数在允许范围内。
- 配置版本与主控运行命令携带的版本一致。

## 同步状态

主控侧从站配置同步状态：

| 状态 | 含义 | 是否允许运行 |
|---|---|---|
| `unknown` | 主控尚未确认从站当前配置 | 否 |
| `pending` | 主控配置已变更，尚未同步到从站 | 否 |
| `synced` | 主控和从站配置版本一致 | 是，还需满足在线、无故障、已校准 |
| `failed` | 最近同步失败 | 否 |
| `invalid` | 配置本身不合法 | 否 |

从站侧配置状态：

| 状态 | 含义 | 输出要求 |
|---|---|---|
| 无配置 | 从站没有可用安全配置 | 全部输出关闭 |
| 配置无效 | 字段冲突、版本不支持或校验失败 | 全部输出关闭 |
| 配置有效 | 可用于校验运行命令和本地安全停机 | 空闲时全部输出关闭 |
| 运行中配置异常 | 运行中发现配置不可信 | 立即安全停机 |

## 持久化语义

当前阶段只定义持久化语义，不规定实现方式。

从站安全配置的持久化必须满足：

- 从站断电重启后不能误恢复运行。
- 如果配置不存在或不可信，从站进入 `UNCONFIGURED`。
- 如果配置损坏，从站不能启动阀门。
- 如果主控配置版本和从站版本不一致，从站拒绝运行命令。
- 如果运行中发现配置异常，从站安全停机。

当前阶段不规定：

- 是否双备份。
- 是否日志式。
- 是否使用 EEPROM/IAP。
- 具体地址。
- 具体文件头。
- 具体写入算法。

这些属于实施阶段的存储可靠性设计。

## 与运行命令的关系

`START_ZONE` 不携带完整安全配置。

运行命令只携带本次运行变化的信息：

- `run_id`
- 配置版本。
- 水路号。
- 运行时长。
- 本次流量保护参数。

从站收到运行命令时必须校验：

- 自身有有效安全配置。
- 运行命令中的配置版本与本地配置一致。
- 目标水路已启用。
- 目标水路不是主阀。
- 当前从站空闲。
- 当前没有禁止运行的故障或锁定。

## 配置变更边界

安全相关配置变更时：

- 主控必须标记该从站配置未同步。
- 主控不能对未同步从站启动运行。
- 从站运行中不接受会改变安全配置的命令。
- 用户必须能在 Web 上看到“配置未同步”。

安全相关配置包括：

- 输入用途。
- 流量计输入。
- 低液位输入。
- 主阀。
- 分支水路启用位图。
- 自吸泵启用。
- 阀门安全参数。
- 通信超时。
- 最大运行时间。
- 电压和流量保护参数。

## 故障语义

配置相关故障分为：

| 故障 | 触发 | 用户处理 |
|---|---|---|
| `ERR_CONFIG_MISMATCH` | 主控命令版本和从站配置版本不一致 | 重新同步配置 |
| `ERR_NOT_CONFIGURED` | 从站没有有效安全配置 | 下发安全配置 |
| `ERR_BAD_VALUE` | 配置字段非法 | 修改 Web 配置 |
| `ERR_INPUT_CONFLICT` | 输入角色冲突 | 修改输入配置 |
| `ERR_OUTPUT_CONFLICT` | 输出角色冲突 | 修改水路/主阀配置 |
| `FAULT_CONFIG_INVALID` | 从站本地配置不可信 | 重新同步或维护 |
| `FAULT_STORAGE_ERROR` | 持久化层报告保存或读取失败 | 进入实施阶段评估存储可靠性 |

配置类故障不能通过普通故障清除绕过。必须重新配置、重新同步或维护处理。
