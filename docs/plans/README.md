## 基础库边界

- ESP32 主控使用 `Esp32Base`。
- STC8H 从站使用 `/Users/tyg/dir/codex_dir/Stc8hBase`。
- 实现必须优先复用基础库已有能力。
- 如果基础库缺少通用能力，不在业务项目里绕路补丁，应先沉淀到基础库。
- Web/API 风格必须匹配 `Esp32Base` 当前能力：固定路径 + `?key=value` query 参数；不采用 REST 路径参数、请求体 JSON 或通用透传接口。
- 从站固件优先稳定、简洁、健壮，业务变化尽量由主控配置和运行命令吸收；从站 OTA 使用 `Stc8hBase` 的 STC8H8K64U 专用 IAP bootloader OTA 基础能力。
