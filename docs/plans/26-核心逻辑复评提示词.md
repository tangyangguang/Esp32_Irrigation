# 26 核心逻辑复评提示词

状态：当前阶段复评工具。本文用于在 `25-核心运行与增强保护降级原则.md` 之后，重新评估当前方案是否真正做到“核心运行逻辑最小、必要、可运行；增强能力不阻断核心执行”。

## 使用方式

把下面提示词复制给一个新的评估会话，要求它只评估方案逻辑、模型、概念、边界、状态、流程和算法，不写代码、不写实施计划、不修改基础库。

## 复评提示词

```text
你是资深嵌入式产品架构师、硬件接口审核工程师和灌溉控制系统软件负责人。

请只评估当前 ESP32 主控 + STC8H 从站灌溉控制系统方案是否已经把“核心运行逻辑”和“增强保护能力”区分清楚。

不要写代码。
不要写实施计划。
不要决定工程目录、文件格式、EEPROM 算法、基础库 API 或具体实现细节。
不要因为增强功能未配置、未学习、未校准，就把核心灌溉运行错误地判为不可运行。
不要为了简化方案而取消必要的安全停机。

项目背景：
- 主控是 ESP32，使用 Esp32Base。
- 从站是 STC8H，使用 Stc8hBase。
- 一个从站对应一个真实水源。
- 一个从站只支持一个流量计。
- 一个从站最多 8 路分支水路。
- 多个从站可以并行运行。
- 同一从站内部同一时间只运行一路水路。
- Web 是完整操作入口。
- LCD 和本地按键只做最小现场辅助。
- 旧代码、legacy 和旧文档只能作为背景，不能作为新版依据。
- 当前只评估 docs/plans/ 下的新版方案文档。

核心原则：
- 核心能力决定能否执行最基本灌溉运行。
- 增强能力用于提前发现异常、提升计量精度、改善维护体验。
- 增强能力未完成配置、学习或校准时，应降级为“不启用该增强能力”，不得阻断核心灌溉运行。
- 增强能力只有在已经检测到明确危险条件时，才可以阻断运行或触发停机。
- Web 必须区分“保护未启用”和“设备故障/禁止运行”。

重点评估：
1. 当前文档是否明确区分核心运行条件和增强保护条件。
2. 标准流速未学习时，是否允许手动和计划运行，只是不启用低/高流量保护。
3. 流量计计量校准未完成时，是否允许运行，只是水量估算不可用或仅供参考。
4. 电压检测未校准时，是否允许运行，只是不启用电压告警和电压停机保护。
5. 已经启用并确认异常的低液位、无水、高流量、电压严重异常，是否仍能阻断运行或本地停机。
6. Web 是否清楚显示哪些增强保护未启用，而不是显示成故障或不可运行。
7. 运行记录是否能记录本次运行缺少哪些增强保护。
8. 自动计划是否遵守同一从站单任务、多从站并行。
9. 多计划同一从站冲突策略是否成熟、可解释、不过度设计。
10. 多计划同一从站冲突是否在保存计划时预警，在触发时清楚记录跳过/冲突原因。
11. 是否仍有“未校准/未学习就禁止运行”的旧口径残留。
12. 是否仍有“可以进入实施计划”的过早结论。
13. 是否存在当前产品范围外的复杂能力，例如云、App、脚本规则、多主控、从站主动上报、Web 任意透传 RS485 帧。
14. 是否存在具体工程实现提前固化，例如工程目录、文件格式、NVS key、EEPROM 地址、类名、函数名、任务名。

输出格式：

一、总体判断
- 当前方案是否已经满足“核心运行最小可用，增强保护可降级”的原则。
- 是否可以继续完善方案，还是可以进入实施计划。
- 如果不能进入实施计划，阻塞的是哪些方案逻辑问题。

二、当前阶段必须修正的问题
按 P0/P1/P2 输出，每项包括：
- 问题描述
- 涉及文档
- 为什么属于方案层问题
- 推荐修正方式

三、增强能力降级检查表
逐项判断：
- 标准流速未学习
- 流量计计量未校准
- 电压检测未校准
- 阀门 PWM 降功率未验证
- SHT30 不可用
- LCD 不可用
- 待机漏水增强判断未启用

四、必要安全停机检查表
逐项判断：
- 低液位已触发
- 启动无水
- 运行中无水
- 高流量
- 通信超时
- 运行超时
- 已校准电压检测发现严重异常
- 配置无效
- 地址疑似冲突

五、多计划同站冲突评估
- 当前默认策略是否成熟。
- 是否会造成用户误解。
- 是否需要更简单或更可解释的策略。
- 是否存在过度设计。

六、过度设计和提前实现细节
- 列出任何不属于当前产品范围的复杂能力。
- 列出任何过早决定的工程实现细节。

七、最终建议
- 继续完善方案 / 可以进入实施计划
- 如果继续完善，下一轮先改哪些文档。
```

## 当前复评通过标准

复评通过必须同时满足：

- 核心运行前提少而明确。
- 增强能力未就绪不阻断核心运行。
- 明确危险条件仍会阻断或停机。
- Web 能解释“保护未启用”和“禁止运行”的区别。
- 多计划同站冲突行为可解释。
- 没有过早进入实施计划的结论。
