# 01 产品范围与硬件边界

## 产品模型

- 系统由 ESP32 主控板和一个或多个 STC8H 从站板组成。
- 主控板只做主控、显示、Web/API、计划调度、配置管理、记录和 RS485 主站通信。
- 从站板也称区域板或执行板，一个从站对应一个真实水源。
- 一个从站只支持一个流量计。如果有第二个水源，应通过 RS485 增加第二个从站。
- 一个从站最多支持 8 路电磁阀输出。
- 8 路阀可全部作为分支水路，也可以选择其中 1 路作为主阀，其余作为分支水路。
- 从站地址由 4 位拨码开关设定，地址范围 `1..15`，不采用配对方式。
- 主控通过 RS485 轮询和控制从站；从站不主动抢占总线上报。

外部端子板、焊点和现场接线板只属于硬件连接边界，不作为软件功能范围。

## 主控板启用范围

主控板启用：

- 12V 输入供电，板上转换到 5V 和 3.3V。
- ESP-WROOM-32E。
- RS485 芯片和方向控制。
- 主控外接按键接口 `H2`：`BTN_OK`、`BTN_ESC`、`BTN_UP`、`BTN_DOWN`。
- 两个软件状态灯：`IO27_LED`、`IO14_LED`。
- LCD2004，接 5V I2C。
- 外接 SHT30 温湿度传感器，接 5V I2C。
- PCA9306 I2C 电平转换。
- DS3231 RTC 作为可靠时间源。

主控板当前不启用：

- ADS1115。
- FRAM。
- 主控侧 ADC、电压检测、NTC、模拟输入。
- 主控侧流量或传感器输入。
- 主控侧 2 路 NMOS 电磁阀输出。
- 主控侧 EG27324 阀驱动。
- 主控侧干接点继电器。
- 主控板作为独立小型灌溉设备的能力。

## 主控板关键 IO

| 功能 | 网表信号 | 当前用途 |
|---|---|---|
| RS485 方向 | `485_DIR` | 启用 |
| RS485 接收 | `485_RX` | 启用 |
| RS485 发送 | `485_TX` | 启用 |
| ESP32 I2C | `I2C_SCL`、`I2C_SDA` | DS3231、PCA9306 后级 LCD/SHT30 |
| 5V I2C | `SCL2`、`SDA2` | LCD2004、SHT30 |
| 主控按键 | `BTN_OK`、`BTN_ESC`、`BTN_UP`、`BTN_DOWN` | 最小本地辅助 |
| 状态灯 | `IO27_LED`、`IO14_LED` | 软件状态指示 |
| RTC 中断 | `IO4_INT` | 可用但第一版不依赖 |

共享 I2C 总线上存在的 ADS1115、FRAM 不因硬件连接存在而进入软件访问范围。

## 主控接口边界

| 接口 | 当前产品边界 |
|---|---|
| `CN1` | 12V 电源输入 |
| `H1` | 3.3V I2C/电源引出，当前不作为 LCD/SHT30 主接口 |
| `H2` | 外接主控按键 |
| `H3` | 主控预留模拟/NTC 接口，当前不启用 |
| `H4` | 主控预留脉冲/传感接口，当前不启用 |
| `U2` | ESP32 调试接口，不作为业务接口 |
| `U3` | 5V I2C 外设接口，用于 LCD2004/SHT30 |

## 从站板启用范围

从站板启用：

- STC8H8K64U-45I-LQFP48。
- RS485 从站通信。
- 4 位拨码开关地址。
- 8 路电磁阀输出。
- `IN1..IN4` 四路数字输入。
- 必须配置 1 路流量计输入。
- 可选配置 1 路低液位输入。
- 可选配置 1 路干接点输出，用于外接自吸泵或水泵继电器。
- 系统电压 ADC 检测。
- 3 个软件状态灯，另 1 个电源灯常亮不由软件控制。
- 默认启用 1 个本地维护/识别按键。

从站板当前不启用：

- 预留 I2C 接口 `H6`。
- 预留 ADC `H8/H9`。
- `SPARE_*` 资源。
- 第二个本地按键的业务功能。

## 从站板关键 IO

| 功能 | 网表信号 | 当前用途 |
|---|---|---|
| RS485 方向 | `485_DIR` | 启用 |
| RS485 接收 | `485_RX` | 启用 |
| RS485 发送 | `485_TX` | 启用 |
| 地址拨码 | `DIP_ADDR0..3` | 启用 |
| 输入 | `IN1..IN4` | 配置流量计、低液位或预留占位 |
| 系统电压 | `ADC_VIN` | 启用，需校准 |
| 干接点输出 | `DRY_OUT1`、`RELAY1A/B` | 可选自吸泵控制 |
| 状态灯 | `LED1..LED3` | 软件状态灯 |
| 维护按键 | `BTN1` | 启用 |
| 阀驱动使能 | `DRV_SD` | 启用，安全态必须关闭 |
| 阀输出 | `PWM1..PWM8`、`VALVE1-..VALVE8-` | 启用 |

## 从站接口边界

| 接口 | 当前产品边界 |
|---|---|
| `DC1` | 从站电源输入 |
| `H1` | 干接点输出，可选控制外接泵继电器 |
| `H2`、`H3` | 5V/GND 引出，不作为软件 IO |
| `H4` | `IN1..IN4` 四路数字输入 |
| `H5` | RS485 A/B |
| `H6` | 预留 I2C，当前不启用 |
| `H7` | STC ISP/复位接口 |
| `H8`、`H9` | 预留 ADC，当前不启用 |

## 职责划分

主控负责：

- Web/API。
- LCD2004 最小本地显示。
- SHT30 温湿度读取。
- RTC 时间。
- RS485 主站通信。
- 从站配置、计划、运行记录、告警和事件。
- 多从站并行调度和同从站冲突跳过。

从站负责：

- 拨码地址读取。
- RS485 从站协议。
- 输入用途映射。
- 阀输出、主阀、自吸泵输出。
- PWM 吸合和保持。
- 单路流量计脉冲计数。
- 低液位检测。
- 系统电压检测。
- 本地安全停机。

## 核心硬件规则

- 一个从站必须配置且只能配置 1 路流量计。
- 低液位可选。
- 自吸泵可选。
- 主阀可选。
- 自吸泵和低液位互不强制依赖。
- 同时配置自吸泵和低液位时，启动自吸泵前必须确认低液位未触发。
- 预留硬件可以存在于网表和 BOM 中，但当前软件不得把它们作为必需能力。
