# 05 系统参数与配置边界

## 基本原则

“系统参数”只承载系统级参数，不承载水源级和水路级配置。

配置按作用域分三类：

- 系统参数：影响整个主控、所有水源或全局保护规则。
- 水源配置：作用于一个从站/水源。
- 水路配置：作用于一个分支水路。

Web 页面必须按这些作用域组织，不再把所有低频项都塞进一个“设置”页，也不拆成高级、维护、日常等多套设置。

## 系统参数

系统参数包含：

- 系统时间和 RTC 同步。
- 主阀和水泵启停时间默认值。
- 流量保护默认参数。
- 低/过流量默认动作。
- 最大单次运行时长，默认 120 分钟，适用于计划和手动运行。
- RS485 通信速率。
- 从站离线判定参数。
- 从站运行通信安全超时规则。

流量保护默认参数见 `03-运行状态与保护规则.md`。

系统参数页按分组正常展示可配置项：

- 时间参数：系统时间、RTC 同步状态和同步操作。
- 全局运行参数：最大单次运行时长、主阀和水泵启停时间。
- 通信参数：RS485 通信速率、从站轮询周期、离线失败次数、运行通信安全超时规则。
- 流量保护参数：启动宽限、无流量确认、漏水检测、低/过流量比例和动作。

这些参数都是当前产品的正常配置项，不因为技术性较强而隐藏；原型只需要避免和水源级、水路级配置混在同一个表单里。

### 当前已确认系统参数清单

| 参数 | 当前结论 |
| --- | --- |
| 系统时间和 RTC 同步 | 需要配置和显示 |
| 最大单次运行时长 | 默认 120 分钟，约束计划和手动运行 |
| 主阀提前开启时间 | 默认 1 秒，非负秒数 |
| 主阀延后关闭时间 | 默认 1 秒，非负秒数 |
| 水泵启动延后时间 | 默认 1 秒，非负秒数 |
| 水泵停止提前时间 | 默认 1 秒，非负秒数 |
| 启动流量宽限时间 | 默认 10 秒 |
| 无流量确认时间 | 默认 10 秒 |
| 漏水检测窗口 | 默认 30 秒 |
| 漏水判定脉冲数 | 默认 3 个脉冲 |
| 低/过流量持续时间 | 默认 20 秒 |
| 低流量比例 | 默认 30% |
| 过流量比例 | 默认 200% |
| 低/过流量默认动作 | 默认仅提示，可改为停机 |
| RS485 通信速率 | 可配置，具体选项实现阶段确认 |
| 从站轮询周期 | 可配置，默认值实现阶段确认 |
| 从站离线失败次数 | 可配置，默认值实现阶段确认 |
| 从站运行通信安全超时 | 必须存在；可由通信参数推导或固定默认值，原则是运行中失联后从站本地安全停机 |

## 水源配置

水源配置作用于一个从站，也就是一个真实水源。

每个水源包含：

- 水源名称。
- 启用/停用。
- 从站地址显示。
- 主阀是否启用。
- 主阀使用哪一路阀；启用主阀时默认阀 8。
- 水泵/自吸泵输出是否启用。
- 流量计输入；默认 `IN1`。
- 低液位输入是否启用。
- 低液位输入；启用后默认 `IN2`。
- 低液位常开/常闭。
- 低液位去抖时间。
- 阀 PWM 参数；水源级统一参数，不逐水路配置。
- 电压读数显示。

水源配置按分组正常展示可配置项：

- 基础信息：水源名称、启用状态、从站地址显示。
- 阀和泵：主阀是否启用、主阀阀号、水泵/自吸泵输出是否启用。
- 输入映射：流量计输入、低液位输入是否启用、低液位输入、常开/常闭和去抖时间。
- 阀驱动：从站级统一 PWM 参数。
- 状态读数：电压读数只显示，不参与告警或保护。

### 当前已确认水源配置清单

| 参数 | 当前结论 |
| --- | --- |
| 水源名称 | 可配置 |
| 水源启用状态 | 启用或停用；只影响新任务 |
| 从站地址 | 只显示，来自拨码硬件，不通过 Web 修改 |
| 主阀是否启用 | 可配置 |
| 主阀使用阀号 | 默认阀 8，可配置 |
| 水泵/自吸泵输出是否启用 | 可配置，从站干接点输出 |
| 流量计输入 | 默认 `IN1`，可从 `IN1..IN4` 选择 |
| 低液位输入是否启用 | 可配置 |
| 低液位输入 | 启用后默认 `IN2`，可从 `IN1..IN4` 选择 |
| 低液位常开/常闭 | 可配置 |
| 低液位去抖时间 | 可配置，默认值实现阶段确认 |
| 阀 PWM 参数 | 水源级统一参数，不逐水路配置；具体数值实现阶段结合驱动确认 |
| 电压读数 | 只显示，不告警、不保护、不停机 |

边界：

- 从站地址来自拨码硬件，Web 只显示和识别，不用 Web 修改拨码地址。
- 电压只显示，不告警、不保护、不停机。
- `IN3/IN4` 只作为流量计或低液位的可选输入，不做通用 IO。
- 从站干接点只作为水泵/自吸泵输出，不做通用继电器。

## 水路配置

水路配置作用于分支水路。

每条水路包含：

- 水路名称。
- 启用/停用。
- 参考流量。
- 校准状态。

### 当前已确认水路配置清单

| 参数 | 当前结论 |
| --- | --- |
| 水路名称 | 可配置 |
| 水路启用状态 | 启用或停用；只影响新任务 |
| 参考流量 | 来自水路校准或正常记录采用 |
| 校准状态 | 显示未校准或已校准 |

边界：

- 水路不选择阀号；阀号由从站阀输出规则和主阀配置推导。
- 参考流量可由水路校准生成，也可从正常浇水记录中的稳定流速一键采用。
- 手动浇水时长不作为水路配置项；由用户本次输入、上次手动时长或计划步骤带入。
- 计划中的浇水时长写在计划步骤里。

## 运行中修改限制

为了避免状态混乱：

- 水源运行中，不能修改该水源的主阀、流量计、低液位和水泵映射。
- 水源运行中，不能修改该水源正在运行水路的启用状态。
- 修改计划不影响已经展开的本次任务。
- 停用水源或水路只阻止新任务，不自动停止正在运行的任务。

停止正在运行的任务必须使用明确的停止操作。

## 当前不设计

- 任意 IO 配置。
- 脚本联动。
- 通用继电器。
- 多角色权限。
- 配置模板。
- 云端配置同步。
- 每个水路单独 PWM。
- 每个水源单独低/过流量比例。
