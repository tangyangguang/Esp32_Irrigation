# 文档03：人机交互设计说明书
## 1. 状态LED指示定义（单色蓝色）
| 状态 | 闪烁模式 |
|------|----------|
| 系统正常运行，WiFi已连接 | 长亮 |
| WiFi连接中/待配网 | 慢闪（1次/秒） |
| 系统异常（水流异常、过流、电磁阀故障） | 快闪（5次/秒） |
| 系统休眠 | 微亮或熄灭 |

## 3. 按键操作逻辑
| 按键 | 操作 | 功能 | 生效范围 |
|------|------|------|----------|
| 编码器 | 顺时针旋转 | 菜单切换向下、数值加 | 所有界面 |
| 编码器 | 逆时针旋转 | 菜单切换向上、数值减 | 所有界面 |
| 编码器 | 短按（<1s） | 确认选择、进入子菜单 | 所有界面（锁定状态下无效，紧急停止除外） |
| 编码器 | 长按（>3s） | 锁定/解锁按键 | 所有界面 |
| 红色停止键 | 短按 | 立即关闭所有电磁阀，停止浇水 | 所有状态下始终有效，包括锁定 |
| 绿色启动键 | 短按 | 快速进入手动浇水设置界面 | 主界面下有效 |
| 组合键 | 红键+绿键+编码器长按5s | 触发恢复出厂确认界面 | 所有状态下有效 |

## 4. Web页面结构
| 页面路径 | 功能 | 核心内容 |
|----------|------|----------|
| / | 首页 | 系统状态、各路状态、当前时间、WiFi状态、剩余浇水时间 |
| /manual | 手动控制 | 选择路数、设置时长、启动/停止浇水 |
| /timer | 定时任务 | 增删改查定时任务，设置循环模式、触发时间、路数、时长 |
| /config | 参数配置 | WiFi设置、管理密码、免打扰时段、天气联动开关、休眠时间设置 |
| /calibrate | 流量校准 | 每路校准系数设置，傻瓜校准/进阶校准切换 |
| /records | 浇水记录 | 历史浇水记录列表、按时间/路数筛选、导出CSV |
| /logs | 系统日志 | 操作日志、系统日志、异常日志列表、导出CSV |
| /reset | 系统重置 | 恢复出厂设置选项、确认逻辑 |

## 5. API接口定义
### 5.1 鉴权说明
所有API调用必须携带鉴权信息，两种方式二选一：
1. HTTP Header方式：请求头中增加`X-API-Key: 你的API密钥`
2. URL参数方式：请求URL后增加`?api_key=你的API密钥`
密钥验证不通过返回`403 Forbidden`错误，默认密钥为`123456`，可在Web配置页面修改。

### 5.2 全量原子API列表
| 类别 | API路径 | 方法 | 参数 | 返回值 | 功能 |
|------|---------|------|------|--------|------|
| **查询类** | /api/v1/status | GET | 无 | 系统状态JSON（时间、WiFi、各路状态、剩余时长、流量） | 获取系统完整实时状态 |
| | /api/v1/records | GET | limit=N, offset=N | 浇水记录JSON数组 | 分页查询历史浇水记录 |
| | /api/v1/logs | GET | type=all/operate/error, limit=N | 日志JSON数组 | 查询系统日志，支持按类型筛选 |
| | /api/v1/config | GET | 无 | 所有配置参数JSON | 获取全部系统配置 |
| **控制类** | /api/v1/water/start | POST | road=X&duration=N | 操作结果JSON | 启动指定路浇水N分钟（road=0为所有路，1-4为对应路） |
| | /api/v1/water/stop | POST | road=X | 操作结果JSON | 停止指定路浇水（road=0为停止所有） |
| | /api/v1/water/delay | POST | minutes=N | 操作结果JSON | 所有运行中的浇水任务延长N分钟 |
| | /api/v1/skip/today | POST | 无 | 操作结果JSON | 跳过今日所有定时任务 |
| | /api/v1/skip/tomorrow | POST | 无 | 操作结果JSON | 跳过明日所有定时任务 |
| | /api/v1/skip/days | POST | days=N | 操作结果JSON | 跳过未来N天的所有定时任务 |
| | /api/v1/skip/cancel | POST | 无 | 操作结果JSON | 取消所有跳过设置，恢复正常定时 |
| **配置类** | /api/v1/config/set | POST | key=xxx&value=xxx | 操作结果JSON | 修改单个配置参数 |
| | /api/v1/config/save | POST | 配置JSON | 操作结果JSON | 批量修改所有配置参数 |
| | /api/v1/config/reset | POST | 无 | 操作结果JSON | 恢复配置到默认值（保留记录日志） |
| **维护类** | /api/v1/system/reboot | POST | 无 | 操作结果JSON | 重启系统 |
| | /api/v1/system/reset | POST | type=config/all | 操作结果JSON | 恢复出厂：type=config仅恢复配置，type=all恢复配置+清空记录 |
| | /api/v1/system/ota | POST | firmware=文件流 | 升级进度JSON | OTA升级固件 |
