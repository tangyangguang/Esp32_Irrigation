# ESP32 灌溉控制器固件

本目录是新固件的实现位置。代码从当前确认方案重新建立，不继承旧固件。

后续实现只依据：

- `../docs/当前方案/` 中确认的软件设计。
- `../pcb_irrigation/` 下的定稿 BOM 和网表。
- `../../Esp32Base` 已核实的公共能力。

当前已建立 PlatformIO 工程、业务配置默认值、定稿 PCB 引脚定义、硬件安全层和应用启动骨架。后续按 `docs/当前方案/03-软件设计原则与边界.md` 分层实现，不能恢复已经删除的旧实现。

常用验证命令：

```sh
pio test -e native
pio run -e esp32_irrigation
```
