# Web 原型说明

本目录存放 ESP32 灌溉项目的 Web 页面原型。原型用于确认页面结构、操作流程、字段和中文文案，不承载真实业务逻辑，不接入 PlatformIO 构建。

## 当前版本

本版按当前固件页面能力重新设计，导航保持为：

`灌溉 / 状态 API / 手动 / 计划 / 配置 / 记录`

## 页面清单

- `index.html`：原型入口和评审状态。
- `dashboard.html`：首页，展示设备状态、阀门状态、异常提示和后续计划。
- `status-api.html`：状态 API 结构示意。
- `manual.html`：手动启动、停止某一路、停止全部和运行时状态。
- `plans.html`：今日/明日计划预览和 8 个计划列表。
- `plan-edit.html`：独立计划编辑页。
- `settings.html`：业务配置、认证入口和恢复出厂。
- `records.html`：浇水记录、事件记录和导出入口。

## 设计原则

- 中文界面，简洁大方，电脑宽屏多列，手机单列展示。
- 所有改变状态的操作都必须有确认提示。
- `/esp32base` 已提供 WiFi、认证账号密码、OTA、日志、重启等系统工具，业务页面只保留必要入口。
- 原型保持轻量静态 HTML/CSS，不引入前端框架，后续应能映射到 ESP32 服务端 HTML。
- 页面不追求装饰感，优先清晰、稳定、好用。
