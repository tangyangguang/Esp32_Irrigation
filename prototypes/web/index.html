灌溉控制器 Web 原型

当前真实页面以服务端 HTML 为准：手动浇水已合并到首页，计划按 4 路 × 每路 6 条组织。

原型说明

对应导航首页 / 近期计划 / 历史记录 / 计划配置 / 灌溉设置
首页职责显示设备状态、4 路浇水状态、手动启动、停止操作和当前告警
交互边界所有改变状态的操作均需确认
实现方式静态 HTML/CSS，不接入固件

设计原则

不做复杂 SPA，不堆装饰。顶部导航负责去向，总览页不再重复放快捷入口。

页面清单

页面功能原型
首页设备状态、4 路浇水状态、手动启动、停止操作和异常提示打开
近期计划今天、明天、后天的计划执行状态和计划触发结果，可跳过某次计划打开
历史记录只显示浇水任务记录打开
计划配置计划列表和修改入口打开
编辑计划单一路计划的执行时间、循环规则和目标时长打开
灌溉设置水路启用、名称、固定引脚、流量和安全阈值打开
