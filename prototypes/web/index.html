灌溉 状态 API 手动 计划 配置 记录

灌溉控制器 Web 原型

按当前固件页面能力重新设计，简洁、响应式、中文界面。

原型说明

对应固件导航灌溉 / 状态 API / 手动 / 计划 / 配置 / 记录
页面目标电脑宽屏可多列浏览，手机单列清晰操作
交互边界所有改变状态的操作均需确认
实现方式静态 HTML/CSS，不接入固件

设计原则

不做复杂 SPA，不堆装饰。首页看状态，手动页做操作，计划编辑进独立页，系统能力通过 Esp32Base 入口提供。

页面清单

页面功能原型
首页设备状态、阀门状态、异常提示、后续计划打开
状态 API展示当前 JSON 状态结构打开
手动启动浇水、停止某一路、停止全部、运行时状态打开
计划今日/明日计划预览、8 个计划列表打开
编辑计划独立计划编辑页面打开
配置基础、每路、水流、按键、认证、维护打开
记录浇水记录、事件记录、CSV 导出打开
